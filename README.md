## DiscourseNuke

Goal is to create a lightweight token model for bluesky that achieve extremely fast filtering based on natural language requests. So "No more US politics". Current iteration was vibecoded in a day. 

### Future additions 
ADd loop to create topic centroids. Pull more representative feed. Finish the codebook so that the filtering can be done more quickly. Contruct the quantizer such that its optimized for human filtering.
